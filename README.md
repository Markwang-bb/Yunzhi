## 安装

确保你已安装 Node.js，然后运行以下命令：

```bash

npm install -g yunzhi 

```


## 使用方法

基本用法：

```bash

yunzhi -c < 城市名称 >

```


更多选项请运行：

```bash

yunzhi --help

```
